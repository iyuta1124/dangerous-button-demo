📝 Tampermonkey防止スクリプト

以下のスクリプトで上のボタンを消すことができます：

// ==UserScript== // @name 本番破壊防止 // @match * // @grant none // ==/UserScript== (function() { 'use strict'; // 危険なボタンを探して隠す document.querySelectorAll('button').forEach(button => { const text = button.textContent || ''; if (text.includes('破壊') || text.includes('削除') || button.classList.contains('btn-danger')) { button.style.display = 'none'; console.log('危険なボタンを隠しました:', text); } }); })();
使い方：
1. Tampermonkey拡張機能をインストール
2. 上記コードを新しいスクリプトとして登録
3. このページをリロードして確認
